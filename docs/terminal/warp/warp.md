# Warp

[Warp](https://warp.dev/) is a modern, GPU-accelerated terminal emulator with built-in AI assistance, command history search, and block-based output. It is the primary terminal used for all homelab work.

## Installation

Download and install from [warp.dev](https://warp.dev/). No Homebrew formula is needed — Warp ships as a native macOS app.

## Why Warp

- Block-based output makes it easy to copy individual command results without selecting around prompts
- Built-in command search and history filtering
- Works well with Starship for a rich prompt (see [Starship](../starship/starship.md))
